# FF2 - เว็บขายรูปนางแบบ

แพลตฟอร์มออนไลน์สำหรับนางแบบและผู้ขายรูปภาพ

## ฟีเจอร์หลัก

### ระบบสมาชิก
- สมัครสมาชิกด้วย Email
- สร้างได้สูงสุด 50 โปรไฟล์ต่อ 1 Email
- บังคับกรอก LINE ID

### ระดับสมาชิก
- **Super** - แสดงช่องที่ 1 (5 บาท/วัน)
- **Model** - แสดงช่องที่ 2 (5 บาท/วัน)
- **ผู้ใช้ทั่วไป** - แสดงช่องที่ 3 (ฟรี - หลังอนุมัติโพสต์)

### ระบบโพสต์
- ใส่รูปได้ 3-5 รูปต่อโพสต์
- รองรับไฟล์ .jpg, .png
- ต้องผ่านการยืนยันจาก Admin

### ระบบเติมเงิน
- โอนเงินผ่านธนาคาร
- ส่งสลิปเพื่อยืนยัน
- Admin ตรวจสอบและอนุมัติ

### ระบบโอนย้ายวันใช้งาน
- โอนวันระหว่างโปรไฟล์ในบัญชีเดียวกัน
- บันทึกประวัติการโอน

## การติดตั้ง

```bash
# ติดตั้ง dependencies
npm install

# สร้างฐานข้อมูล
npx prisma generate
npx prisma db push

# รัน development server
npm run dev
```

## โครงสร้างโปรเจกต์

```
FF2/
├── app/
│   ├── (auth)/          # หน้าล็อกอิน/สมัคร
│   ├── (user)/          # หน้าสมาชิก
│   ├── (admin)/         # หน้าหลังบ้าน
│   ├── api/             # API routes
│   └── page.tsx         # หน้าแรก
├── components/          # Components
├── lib/                 # Utilities & Prisma
└── prisma/              # Database schema
```

## สร้าง Admin แรก

1. สมัครสมาชิกปกติ
2. เปิด Prisma Studio:
   ```bash
   npx prisma studio
   ```
3. แก้ไข role ของ user เป็น "ADMIN"
